================================================================================
                    PDF EXPORT - HOW IT WORKS (CONFIRMED)
================================================================================

The system HAS a built-in PDF export feature. Here's the flow:

1. User clicks "Print" button on invoice
2. PrintPreviewDialog component opens
3. Shows full HTML/CSS preview in iframe
4. Three action buttons appear:
   - "Print to Printer" (for physical printer)
   - "Browser Print Dialog" (alternative)
   - "Save as PDF" ← THIS IS WHAT YOU USE

5. Click "Save as PDF"
6. Browser PDF engine converts HTML to PDF
7. File downloads to Downloads folder
8. Open and inspect visually

Code location: src/shared/components/PrintPreviewDialog.tsx (lines 80-94)
